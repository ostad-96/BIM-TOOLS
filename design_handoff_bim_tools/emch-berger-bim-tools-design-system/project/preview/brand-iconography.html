Iconography — hand-tuned stroked SVG, 24×24, currentColor
No icon font, no library — each page hand-draws SVGs. viewBox="0 0 24 24", fill="none", stroke="currentColor". Stroke widths: 1.8 (decorative), 2.0 (functional buttons), 2.2 (arrows). linecap and linejoin always round. Substitute with Lucide when adding new icons.
